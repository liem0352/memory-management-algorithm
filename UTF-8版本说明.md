## UTF-8版本特点

- **完美中文显示** - 使用UTF-8编码，无乱码
- **自动编码设置** - 程序自动设置控制台编码
- **完整中文界面** - 所有提示信息都是中文
- **功能完整** - 支持三种内存分配算法

## 技术实现

程序通过以下方式解决中文乱码问题：

1. **Windows API设置**：
  ```c
  SetConsoleOutputCP(65001);  // 设置输出编码为UTF-8
  SetConsoleCP(65001);  // 设置输入编码为UTF-8
  ```

2. **本地化环境设置**：
  ```c
  setlocale(LC_ALL, "zh_CN.UTF-8");
  ```

3. **批处理脚本支持**：
  ```batch
  chcp 65001 >nul  // 设置控制台代码页为UTF-8
  ```

## 操作示例

###基本操作流程
```
1. 选择算法：1（首次适应算法）
2. 分配内存：作业1，200KB
3. 分配内存：作业2，300KB  
4. 分配内存：作业3，150KB
5. 查看内存状态
6. 回收内存：作业2
7. 查看内存状态（可以看到相邻分区合并）
```

###算法选择
-**首次适应算法**：按顺序查找第一个足够大的空闲分区
-**最佳适应算法**：查找最小的足够大的空闲分区
-**最坏适应算法**：查找最大的空闲分区

## 程序功能

1. **内存分配** - 为指定作业分配内存
2. **内存回收** - 回收指定作业的内存
3. **分区合并** - 自动合并相邻的空闲分区
4. **状态显示** - 实时显示空闲区表和已分配表
5. **算法切换** - 支持三种分配算法动态切换
